# 📊 JPMorgan Quantitative Research Project

## 🔍 Overview
This project is based on the JPMorgan Chase Quantitative Research Virtual Experience (Forage). It simulates real-world financial modeling tasks including commodity price forecasting, contract valuation, and credit risk analysis.

The goal of this project is to apply data analysis and quantitative techniques to solve practical problems in finance.

---

## ⚙️ Key Tasks & Implementation

### 1️⃣ Commodity Price Analysis
- Analyzed historical natural gas price data
- Built a model to estimate prices for any given date
- Identified seasonal trends using time-series visualization

### 2️⃣ Contract Pricing Model
- Developed a pricing model for natural gas storage contracts
- Incorporated:
  - Injection & withdrawal prices
  - Storage costs
  - Volume constraints
- Calculated net contract value

---

### 3️⃣ Credit Risk Modeling
- Built a model to estimate **Probability of Default (PD)**
- Calculated **Expected Loss (EL)** using recovery rate
- Used borrower financial data for prediction

---

### 4️⃣ FICO Score Bucketing
- Converted continuous FICO scores into categorical risk buckets
- Applied quantile-based discretization
- Estimated default probability per segment
